통합 시뮬레이션: 매월 재고 = 전월재고 + Sell-In(입고) - Sell-Out(판매) 로직으로 적정 재고를 유지하기 위한 필요 발주량(Sell-In)을 자동 계산합니다.
• Sell-Out: 예상 월 판매량 (직접 입력 또는 [자동 채우기]로 과거 데이터 기반 추정)
• 목표 WOS: 유지하고 싶은 적정 재고 주수 (기본값 = 채널별 HEALTHY 기준)
• 결과: 월별 필요 입고량 + 월말 예상 재고 + WOS + 신호등
